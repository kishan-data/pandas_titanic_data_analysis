# pandas_titanic_data_analysis

## 📌 Project Overview

This notebook performs exploratory data analysis (EDA) on the Titanic dataset using **Python and Pandas**.

The goal is to analyze passenger data, apply filtering logic, perform grouping operations, and extract meaningful insights using real-world data manipulation techniques.

---

## 🛠️ Tools & Libraries Used

* Python
* Pandas
* Jupyter Notebook

---

## 📂 Dataset

The dataset contains passenger details such as:

* Passenger Class (`Pclass`)
* Age
* Fare
* Gender (`Sex`)
* Embarked Port
* Survival Status (`Survived`)
* Cabin Information
* Family details (`SibSp`, `Parch`)

---

## 🔎 Key Concepts Covered

### ✅ Data Filtering

* Multiple condition filtering using `&` and `|`
* Handling missing values with `isna()` and `notna()`
* Logical nested filtering

### ✅ GroupBy Operations

* Survival rate by class
* Average fare by age and class
* Aggregations using `mean()`, `sum()`, `count()`
* Advanced `agg()` usage

### ✅ Pivot Tables

* Survival distribution by embarkation port
* Class-wise analysis

### ✅ Sorting & Ranking

* Sorting within groups
* Multi-column sorting
* Identifying top values using `sort_values()` and `head()`


## 📊 Sample Analysis Performed

* Identified which passenger class had the highest survival rate
* Found top ages with highest average fare per class
* Filtered female first-class passengers paying above-average fare
* Analyzed survival distribution by embarkation port
* Extracted deck information from Cabin column using string methods
